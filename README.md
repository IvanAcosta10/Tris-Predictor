# TRIS Predictor V2

1. Descomprime el ZIP.
2. Crea un repositorio público en GitHub.
3. Sube `app.py` y `requirements.txt`.
4. En Streamlit Community Cloud crea una app y selecciona `app.py`.
5. Dentro de la app presiona **Actualizar resultados**.

Para ejecutarlo en Windows:

```powershell
py -m pip install -r requirements.txt
py -m streamlit run app.py
```
